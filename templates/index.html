Funcionalidade

Este site oferece ferramentas para análise de demissões de funcionários na empresa. Com ele, você pode gerar gráficos detalhados e realizar cálculos sobre as demissões realizadas. Utilize a opção "Análise Geral" para obter uma visão ampla das demissões e "Análise Mensal" para revisar os dados detalhados de cada mês. Navegue pelas opções para explorar diferentes aspectos das demissões e obter insights valiosos para a gestão de pessoal.
